# BUILDORA

Sistema de presupuestos de obra y análisis de precios unitarios desarrollado en Python y MySQL.

## Características

- Gestión de clientes
- Gestión de proyectos
- Gestión de materiales
- Gestión de mano de obra
- Gestión de herramientas
- Gestión de APUs
- Gestión de presupuestos
- Gestión de AIU
- Dashboard
- Reportes Excel
- Reportes PDF

## Tecnologías

- Python
- CustomTkinter
- MySQL
- OpenPyXL
- ReportLab

## Arquitectura

- Views
- Controllers
- Services
- Repositories
- Database
